キーボードショートカット一覧

単体キー（テキスト入力中は無効）
F ファイルメニューを開く
H 操作説明を表示/非表示
Ctrl + キー操作
Ctrl + D テキストをすべてリセット
Ctrl + P 選択中のテキストを再生
Ctrl + R 全テキストを連続再生
Ctrl + S 現在行を個別保存
Ctrl + Shift + S 全テキストを連続保存
Ctrl + T リップシンクテストを開始
Ctrl + Z 設定/テキストのUndo
Ctrl + Y 設定/テキストのRedo

💡 Undo / Redo について

Ctrl + Z と Ctrl + Y は状況に応じて動作が変わります：

  • テキスト入力中：テキストの変更を取り消し / やり直し
  • 音声パラメータ・エフェクトタブ：パラメータの変更を戻す / やり直す
  • その他のタブ：動作しない場合があります

※ 複数回の戻る / やり直しに対応しています
