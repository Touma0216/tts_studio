🎙️ TTSスタジオ 操作説明

📁 ファイル操作

ショートカット 機能 説明
Ctrl + F ファイルメニュー モデル読み込み、履歴表示
Ctrl + H 操作説明 この画面を表示

🎵 音声再生

ショートカット 機能 説明
Ctrl + P 現在行を再生 選択中のテキスト行を個別再生
Ctrl + R 連続再生 全てのテキスト行を順番に再生

📝 テキスト編集

ショートカット 機能 説明
Ctrl + N テキスト行追加 新しいテキスト入力行を追加（最大9行）
Ctrl + Enter 改行挿入 テキスト入力中に改行を挿入
Enter 入力確定 テキスト入力を確定してフォーカスを外す

🎯 タブ切り替え

ショートカット 機能 説明
Ctrl + Tab マスタータブ ★タブ（全行に適用される設定）
Ctrl + 1 1番目のテキスト行 1行目のパラメータタブに切り替え
Ctrl + 2 2番目のテキスト行 2行目のパラメータタブに切り替え
... 3〜9番目も同様 Ctrl + 数字で対応行に切り替え

🎭 音声パラメータ

ショートカット 機能 説明
Ctrl + E 感情選択 現在のタブの感情コンボボックスを開く

💾 ファイル保存

ショートカット 機能 説明
Ctrl + S 個別保存 各テキスト行を個別のWAVファイルで保存
Ctrl + Shift + S 連続保存 全テキスト行を1つのWAVファイルに結合保存

⭐ マスターパラメータ機能

使い方の流れ：
  1. Ctrl + Tab で★タブに移動
  2. 基本となる音声パラメータを設定
  3. ★タブの設定が全ての行に自動反映される
  4. 個別の行で微調整したい場合は Ctrl + 1〜9 で個別タブへ
  5. 個別タブの変更は★タブには影響しない
💡 ヒント： テキスト入力中でもCtrlキーを使ったショートカットは動作します。入力を中断することなく、効率的に操作できます。
この画面は背景をクリックするか、再度 Ctrl + H で閉じることができます
