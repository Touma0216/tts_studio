キーボードショートカット一覧

ファイル・ウィンドウ操作
Ctrl + F ファイルメニューを開く
Ctrl + H 操作説明を表示/非表示
編集操作
Ctrl + Z 前の設定に戻す（パラメータ・テキストUndo）
音声再生
Ctrl + P 現在フォーカス中の行を再生
Ctrl + R 全テキストを連続再生
テキスト・タブ操作
Ctrl + N 新しいテキスト行を追加
Ctrl + Tab ★マスタータブへ移動
Ctrl + E 感情選択コンボボックスを開く
Ctrl + 1 - 9 指定のテキスト行にフォーカス
ファイル保存
Ctrl + S 現在行を個別ファイルとして保存
Ctrl + Shift + S 全テキストを1つのファイルに連続保存

💡 Undo機能について

Ctrl + Z は状況に応じて動作が変わります：

  • テキスト入力中：テキストの変更を取り消し
  • 音声パラメータタブ：感情・話速・ピッチなどの前の設定に戻す
  • その他のタブ：動作しません

※ 1回だけ前の状態に戻せます
